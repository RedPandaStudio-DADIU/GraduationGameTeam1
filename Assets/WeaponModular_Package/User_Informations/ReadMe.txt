////////////////////////////////////* Scifi Weapon Modular : *////////////////////////////////////////
//////////////////////////////////////////////////////////////////////////////////////////////////////

Contain : 
	- 5 Weapons
	- 4 Accessories   
	- 4 Butts
	- 4 Cannons
	- 5 Sights

	- 3 Textures per object

	- 1 Tool used to assemble differents weapon parts, and save the result into a prefab.

Tool using order :
	Tool localised in "Weapon Modular" button menu, then "BuilderPreview".
	Select desired weapon, sub-part's weapon and texture.
	Enter a name for your prefab, then click "Create Prefab" (Save in "Weapon Prefab" folder).
	
Warning : 
	Don't break the hierarchy's folder into the package, if you need to use the tool. 
	You can move the entire package where you want in the "Asset" folder of your project.
